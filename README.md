This project is created to see the interaction between Ghost Gatsby and deploy it to Netlify.

_This project use ghost-template-starter_

## Getting started

1. Bring the docker compose up:

   ```sh
   docker-compose up
   ```

2. Bring the Gatsby application up:
   ```js
   yarn install
   yarn dev
   ```
